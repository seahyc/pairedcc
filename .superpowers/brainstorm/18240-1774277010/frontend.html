UI Components

Top bar: Logo, doc title, presence avatars (humans + agents), share button
Cursors: Color-coded per participant, with name label. Agents get a 🤖 indicator.
@-mentions: Inline in the doc flow. Type @ to get autocomplete of connected agents + collaborators.
Agent responses: Appear as styled blocks in the doc. Agent edits show up in real-time with its cursor moving.
